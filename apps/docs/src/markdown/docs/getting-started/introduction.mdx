# Introduction

Welcome to the Hiko documentation!

Hiko is a modern, accessible, and customizable UI library for React. It is built with Tailwind CSS and TypeScript, and is designed to be easy to use and customize.

---

## Installation

### Install

```bash
npm install @hikoui/react @hikoui/theme
```

### Import

```tsx
import { Button } from "@hikoui/react";
```

### Usage

```tsx
<Button>Click me</Button>
```
